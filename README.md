# WalletView

Dapp to watch all your RC-20 
 
# Clone

This repository has submodules, to clone execute this command:
 
 ```
 git clone --recurse-submodules https://github.com/wooltar/walletview.git
 ```
 
## Docker start
```
$ docker-compose up
```

## Run all tests
```https://walletviewbackend.herokuapp.com/
$ yarn test
```
